## ‚ùì Git vs GitHub - Quick Answer

**Git** = Version control software on your computer (tracks changes to files)
**GitHub** = Cloud platform for hosting Git repositories (like Google Drive for code)

Think: Git is Microsoft Word, GitHub is Google Docs

## üÜò If You Get Stuck

**GitHub not updating?**
```bash
git status          # See what's changed
git add .           # Stage everything
git commit -m "Your message"
git push            # Upload to GitHub
```

**Want to try something without breaking things?**
```bash
git checkout -b experiment    # Create a branch
# [Make your changes]
git checkout main             # Go back to main
```

**Made a mistake?**
- Check the flashcards for "undo" commands
- Remember: commits are snapshots - you can always go back
- GitHub keeps your history - nothing is truly lost
